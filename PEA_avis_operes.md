# PEA — Avis d'Opérés (données brutes)

> **Mode d'emploi :**
> Colle ici le contenu de chaque avis d'opéré, dans l'ordre chronologique.
> Pour chaque avis, utilise le bloc structuré ci-dessous.
> Une fois tous les avis renseignés, demande l'analyse complète.

---

## Format attendu pour chaque avis d'opéré

```
--- AVIS D'OPÉRÉ ---
Date              : JJ/MM/AAAA
Type              : ACHAT | VENTE | DIVIDENDE | FRAIS | AUTRE
Titre             : Nom complet du titre
ISIN              : (ex. FR0000131104)
Quantité          : (nombre de parts / actions)
Prix unitaire     : X,XX €
Montant brut      : X,XX €
Frais / Commission: X,XX €
Montant net       : X,XX €
Sens              : DÉBIT | CRÉDIT
Référence ordre   : (optionnel)
---------------------
```
